Данный репозиторий содержит портированный код агента E-Sapa. OilService был заменен SunpApi.
Для запуска необходимо:
1. Заполнить конфигурацию objectSettings.json
2. Сбилдить проект
3. Заменить в install.bat путь до exe-исполняемого файла
4. Запустить install.bat от имени администратора
5. Отслеживать логи в папке с exe по пути /logs/дата_лога.log

Приложение работает следующим образом:
1. Запрос отчета по хранению с КПУ
2. Запрос отчета по поставкам с КПУ
3. Создание задачи с данными хранения/поставки в папку /tasks/имя_файла.json
4. Отправка данных
5. В случае принятия сервером данных, файл с задачей удаляется
6. В случае непринятия данных сервером, повтор через заданное время (макс. 15 повторений)
7. В случае, если количество повторений закончилось, файл задачи удаляется
